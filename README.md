Mineclock
===========

The Pebble watchface looks like the Minecraft clock.

![](https://dl.dropboxusercontent.com/u/8932138/downloads/pebble/mineclock/screenshot_v1.2_chalk.gif)

## Development

```
npm run build
pebble build
# Pebble Classic
npm run aplite
# Pebble Time
npm run basalt
# Pebble Time Round
npm run chalk
# Pebble 2
npm run diorite
```
