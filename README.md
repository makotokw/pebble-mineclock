Mineclock
===========

The Pebble watchface looks like the Minecraft clock.

![](https://dl.dropboxusercontent.com/u/8932138/downloads/pebble/mineclock/screenshot_chalk.gif)

## Development

```
npm run build
pebble build
# Pebble Classic
npm run a
# Pebble Time
npm run b
# Pebble Time Round
npm run c
# Pebble 2
npm run d
```
